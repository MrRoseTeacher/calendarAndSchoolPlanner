User Guide

Application Goals

This calendar is desgined to do the following:

  • Generate a 5-day per week calendar based on a start date and end date.
  • Mark holidays, and input various items. The items may be content, evaluations, links, homework, personal notes, and personal links.
  • Copy the plan for the week to post on Brightspace for students. Items marked as personal notes or personal links are not included in the copy.
  • Drag and drop items between days and within the calendar
  • Save and load in calendars for different classes
  • Merge calendars- use a new blank calendar with new dates and holidays and merge the previous semester's topics

Saving and Loading Calendars

Calendars are saved as .json files. You may keep these files somewhere convenient such as your Google Drive. You may use the website on your mobile phone, and keeping your files in your Google Drive can make this easier. As you make changes, a notification will appear at the top of the page as well as on the bottom (!). You can save by selecting the save button at the top or bottom (S). The filename for your calendar will match what you've entered in as the calendar title. Use the load button to choose a calendar file to load.

Chrome Desktop Browser Recommended: This application use webkit File Handling API to be able to overwrite a calendar file and avoid multiple copies. At this time, browsers such as Safari, Firefox, and mobile cannot support this. When using this application in those settings, you will download a new copy of your calendar file into your downloads folder.

Merging Calendars

To merge a calendar, start by generating a new calendar and mark the new semester holidays. You may also load in a blank semester calendar instead of generating your own. Next, select 'merge' and choose one of your previous calendar files. The resulting calendar will have the content of your previous calendar but will be updated with new dates and holidays. Change the title of your calendar and then save the file.

Copying to Brightspace

After you have copied the HTML code, you can paste the code in a Brightspace page or announcement. When in the text editor, first use your cursor to select where in the content you'd like the calendar week to appear. Then, you must open the "source code" window by selecting the button pictured.

Then, paste your code and select "save" at the bottom of the source code window. In the annoucements section, the calendar will always show up in a single column. In a Brightspace content page, the calendar will appear with 5 columns.

To make the calendar adjust its look on content pages when viewed with a mobile device, add this code to the <head> of the Brightspace page.

<link rel="stylesheet" href="https://school-planner-beta.vercel.app/brightspaceStyles.css">

Input Modes and Keyboard Shortcuts

There are 2 input modes: "Each Time" and "Persistent". In the "Each Time" mode, selecting a day will prompt you each time for which type of content to add. Using "Persistent" mode, you select which item you'd like to add and you may add several of those items without interacting with a prompt each time. Change the item type using the numbered button at the bottom left of the scrren.

You may change the input mode selecting the input button (I) at the bottom left of the screen. Using "Each Time" is recommended for new users and when on a mobile device. The default mode is "Persistent"

You may use the following keyboard shortcuts when the popup prompt is not open:

  • Press "i" to change the input mode
  • Press 1-7 while in persistent mode to change which content item to add next
  • Press "shift + s" to save your calendar
